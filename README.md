## Readme
For all code files please run with Matlab 2023 R2 or higher version.

Function list of all uploaded files:

**Plate_RL_Dispersion.m**: for calculating the dipersion curves of the given plate in certain thickness and certain material.

**Symmetric.m**: auxiliary funtion of **Plate_RL_Dispersion.m**

**bandgapcalculation.m**: for visulize all unit bandgaps.

**solved_xd_newton.m**: newtonian iterative deduction of diffraction foucus calulation (See appendix of the paper).

**transmissionRateCal.m**: for calculate the transmission rate of a selected meta-unit.
